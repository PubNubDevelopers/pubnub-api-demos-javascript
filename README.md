# PubNub JS SDK Demo Code

Be sure to insert your PubNub API Keys.

```javascript
const pubnub = new PubNub({
    publishKey : 'YOUR PUBNUB PUBLISH KEY HERE',
    subscribeKey : 'YOUR PUBNUB SUBSCRIBE KEY HERE'
});
```
